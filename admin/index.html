Guide

This panel lets you edit site content without code.

Quick Start

  1. Login at login.html
  2. Open needed tab (Main, About, ...)
  3. Choose admin UI language (RU/EN/...)
  4. Choose content language on edit page
  5. Load → Edit → Save

Navigation

  • Main Page — main screen (Hero, Welcome, Comfort, Taste, Apartments)
  • About Page — history, features, team
  • Rest-Bar Page — restaurants/bars and their blocks
  • Gallery Page — gallery categories and images
  • Rooms Page — room types list and cards
  • Room Detail — per‑room pages (by slug)
  • Contact Page — contacts, map, form
  • Settings — SEO codes (GSC, Yandex), analytics (GA4, GTM, Pixel)

Editing

  • “Load” pulls values for the chosen language
  • “Save” writes changes to DB (by keys)
  • “Export JSON” downloads current language JSON
  • Show/Hide toggles control section visibility

Images & Upload

  • Use the file button — URL fills automatically after upload
  • Supported: jpg, png, webp, avif, gif. Limit 5 MB
  • To remove, clear field or upload a new one

Languages

  • Top selector = admin UI language
  • “Content language” = language you edit
  • Translations stored by flat keys (hero.greeting).

Preview

  • Open public page with parameters: ?preview&lang=ru
  • Optionally add &draft to overlay local draft if supported
  • Examples: /index.html?preview&lang=ru, /about.html?preview&lang=en

Security

  • Login uses HttpOnly cookie. Use “Logout” after work
  • Uploads accept only images and limited by size

FAQ

  • “Don’t see changes” — refresh with ?preview and right lang; ensure you saved the right language
  • “Save doesn’t work” — re‑login if session expired
  • “Image not shown” — URL must start with /uploads/... and format must be supported
